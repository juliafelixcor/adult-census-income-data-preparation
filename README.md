# Preparação de Dados e Análise Exploratória - Adult Census Income

Este projeto tem como objetivo realizar a preparação e a análise exploratória de dados
utilizando a base Adult (Census Income Dataset), disponível no UCI Machine Learning Repository.

A atividade envolve a análise da qualidade dos dados, tratamento de valores ausentes,
remoção de atributos irrelevantes, aplicação de técnicas de normalização e geração de
visualizações para melhor compreensão do problema.
